# 🚀 GenAI Prompt Engineering – LangChain Assignment

## 📌 Overview

This project is part of the **Data Science Internship at Innomatics Research Labs**.
The goal is to build a **Mini Prompt Engine** using LangChain that generates dynamic and reusable prompts instead of hardcoded ones.

---

## 🎯 Objective

* Understand and use PromptTemplate in LangChain
* Build reusable prompt systems
* Design real-world LLM prompt pipelines
* Implement input validation and structured prompting

---

## ⚙️ Tech Stack

* Python
* LangChain
* Google Colab / Jupyter Notebook

---

## 🧠 Features Implemented

### ✅ Task 1: PromptTemplate Conversion

Replaced hardcoded prompts with reusable templates.

### ✅ Task 2: Multi-Input Prompt System

Handled multiple inputs:

* Topic
* Audience
* Tone

### ✅ Task 3: Prompt Variations

Created multiple prompt styles:

* Teaching
* Interview
* Storytelling

### ✅ Task 4: ChatPromptTemplate

Built role-based chat prompts:

* Teacher
* Interviewer
* Motivator

### ✅ Task 5: Input Validation

Validated:

* Audience → beginner / intermediate / expert
* Tone → formal / casual / fun

### ✅ Task 6: Prompt Generator

Dynamic prompt generation using templates and validation.

### ✅ Task 7: Reusability Test

Used one template with multiple inputs to ensure scalability.

---

## 🔁 Final Pipeline

User Input → Validation → Template Selection → Prompt Generation → Output

---

## 📂 Project Structure

* PromptTemplate implementations
* ChatPromptTemplate system
* Validation module
* Prompt generator engine
* Reusability testing

